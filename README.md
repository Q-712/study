# 图书馆借阅系统系统简介

## 一、系统概述

图书馆借阅系统是一个基于Flask框架开发的Web应用程序，旨在为图书馆提供便捷、高效的图书管理和借阅服务。系统支持多种用户角色，实现了图书的借阅、归还、查询以及用户管理等核心功能，全面提升图书馆的管理效率和用户体验。

## 二、系统架构

### 1. 技术栈
- **后端框架**：Flask (Python)
- **数据库**：SQLite
- **ORM**：Flask-SQLAlchemy
- **用户认证**：Flask-Login
- **前端技术**：HTML5、CSS3、Bootstrap、Jinja2模板引擎

### 2. 系统结构
- **模型层**：定义用户、图书、借阅记录等数据模型
- **视图层**：处理HTTP请求，实现业务逻辑
- **模板层**：使用Jinja2渲染动态页面
- **数据层**：通过SQLAlchemy与SQLite数据库交互

## 三、功能模块

### 1. 用户管理
- **角色划分**：
  - **学生(ROLE_READER)**：可借阅图书、查看借阅历史
  - **管理员(ROLE_ADMIN)**：管理图书、读者和借阅记录
  - **超级管理员(ROLE_SUPER_ADMIN)**：管理管理员账户和系统配置

- **用户功能**：
  - 用户登录/注销
  - 个人信息查看
  - 密码修改

### 2. 图书管理
- **图书录入**：添加新图书信息（书名、作者、出版社、ISBN等）
- **图书查询**：支持按书名、作者、ISBN等条件搜索
- **图书状态管理**：实时更新图书的可借/已借状态
- **图书分类**：支持多维度图书分类管理

### 3. 借阅管理
- **图书借阅**：学生可在线借阅图书，系统自动计算归还日期
- **图书归还**：支持图书归还操作，自动更新借阅记录
- **逾期提醒**：系统自动记录逾期情况
- **借阅历史**：查询所有借阅记录和历史数据

### 4. 系统管理
- **系统配置**：设置借阅期限、逾期规则等参数
- **数据统计**：提供借阅统计报表
- **日志管理**：记录系统操作日志

## 四、系统特点

1. **角色权限分明**：采用RBAC（基于角色的访问控制），确保系统安全
2. **界面友好**：使用Bootstrap框架，界面美观、响应式设计
3. **操作便捷**：流程清晰，操作简单直观
4. **数据安全**：密码加密存储，数据定期备份
5. **可扩展性强**：模块化设计，便于功能扩展和维护

## 五、运行环境

- **操作系统**：Windows/Linux/macOS
- **Python版本**：3.8+
- **依赖包**：详见requirements.txt

## 六、使用说明

### 1. 系统部署
1. 安装依赖：`pip install -r requirements.txt`
2. 启动应用：`python app.py`
3. 访问系统：在浏览器中输入`http://127.0.0.1:5000`

### 2. 初始账号
- 超级管理员：admin001 / 123456
- 管理员：admin002 / 123456
- 学生：reader001 / 123456

### 3. 主要操作流程

#### 图书借阅流程
1. 学生登录系统
2. 搜索并选择要借阅的图书
3. 点击"借阅"按钮完成借阅

#### 图书归还流程
1. 学生登录系统
2. 进入"我的借阅"页面
3. 选择要归还的图书，点击"归还"按钮
4. 确认归还信息，完成归还

#### 图书管理流程
1. 管理员登录系统
2. 进入"图书管理"页面
3. 点击"添加图书"录入新图书信息
4. 可对现有图书进行编辑或删除操作

## 七、系统维护

1. **数据备份**：定期备份instance/library.db文件
2. **系统更新**：根据需求修改代码后，重启Flask服务器即可生效
3. **问题排查**：查看终端输出的日志信息，定位和解决问题

## 八、总结

图书馆借阅系统实现了图书管理的全面信息化，有效提高了图书馆的工作效率和服务质量。系统界面友好、操作便捷、功能完善，能够满足图书馆日常管理的各种需求。未来可根据实际需要，进一步扩展功能，如添加图书推荐、在线续借等功能，提升用户体验。
