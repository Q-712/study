# 图书馆借阅系统

一个基于Flask框架开发的图书馆借阅管理系统，提供学生端、管理端和超级管理员端三个角色入口，实现图书管理、借阅归还、用户管理等核心功能。

## 系统概述

图书馆借阅系统是一个基于Flask框架开发的Web应用程序，旨在为图书馆提供便捷、高效的图书管理和借阅服务。系统支持多种用户角色，实现了图书的借阅、归还、查询以及用户管理等核心功能，全面提升图书馆的管理效率和用户体验。

## 技术栈

- **后端框架**：Flask
- **数据库ORM**：Flask-SQLAlchemy
- **用户认证**：Flask-Login
- **密码安全**：Werkzeug
- **前端框架**：Bootstrap 5
- **模板引擎**：Jinja2
- **数据库**：SQLite

## 系统架构

### 1. 技术栈
- **后端框架**：Flask (Python)
- **数据库**：SQLite
- **ORM**：Flask-SQLAlchemy
- **用户认证**：Flask-Login
- **前端技术**：HTML5、CSS3、Bootstrap、Jinja2模板引擎

### 2. 系统结构
- **模型层**：定义用户、图书、借阅记录等数据模型
- **视图层**：处理HTTP请求，实现业务逻辑
- **模板层**：使用Jinja2渲染动态页面
- **数据层**：通过SQLAlchemy与SQLite数据库交互

## 功能模块

### 1. 用户认证与权限管理
- 用户登录/登出
- 角色权限控制（学生、管理员、超级管理员）
- 密码安全存储与验证

### 2. 学生端功能
- 仪表板：核心功能入口
- 图书借阅：浏览、搜索和借阅图书
- 图书归还：查看和归还当前借阅图书
- 借阅历史：查询所有借阅记录
- 个人信息查看

### 3. 管理端功能
- 仪表板：系统统计数据展示
- 图书管理：添加、删除和查询图书/杂志
- 读者管理：添加和查看学生读者信息

### 4. 超级管理员功能
- 管理员账户管理
- 系统配置管理
- 详细统计分析

## 系统特点

1. **角色权限分明**：采用RBAC（基于角色的访问控制），确保系统安全
2. **界面友好**：使用Bootstrap框架，界面美观、响应式设计
3. **操作便捷**：流程清晰，操作简单直观
4. **数据安全**：密码加密存储，数据定期备份
5. **可扩展性强**：模块化设计，便于功能扩展和维护

## 安装与运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
python app.py
```

访问 `http://127.0.0.1:5000` 进入系统登录页面。

## 核心功能实现

### 1. 图书借阅功能
学生可以浏览可借阅图书，搜索感兴趣的书籍，并执行借阅操作，系统自动计算应归还日期。

### 2. 图书归还功能
学生可以查看当前借阅的图书列表，执行归还操作，系统更新借阅记录和图书状态。

### 3. 图书管理功能
管理员可以添加新图书/杂志，删除未被借阅的图书，查询所有图书信息。

### 4. 数据统计功能
管理端仪表板展示系统统计数据，包括总图书数、总读者数、已借阅图书数、今日借阅量等。

## 安全特性

- 密码使用Werkzeug的hash算法安全存储
- 实现了完善的权限控制，防止越权访问
- 使用ORM参数化查询，避免SQL注入攻击
- 所有敏感操作均需登录验证
